# Data Center Token (DCT)

This repository contains the smart contract implementation for **Data Center Token (DCT)**, developed for the **Decentralised Finance and Blockchain** individual coursework on asset tokenisation design.

## Summary

DCT is implemented as an **ERC-20 fungible token**.  
The repository provides the technical evidence of token creation and deployment, while the full asset-token-market-risk analysis is presented in the written report.

## Token Details

- **Name:** Data Center Token
- **Symbol:** DCT
- **Standard:** ERC-20
- **Contract Address:** `0x1e65896b5481a7af0a929683526507ca7c90d907`

## Purpose of This Repository

This GitHub repository is included as coursework evidence to demonstrate:

- implementation of a fungible token smart contract
- deployment to a test network
- existence of an on-chain contract address
- practical use of blockchain infrastructure in an asset tokenisation context

## Important Note

The repository focuses on the **technical component** of the project.  
Detailed discussion of the following is contained in the report:

- underlying asset description
- rationale for tokenisation
- token design justification
- market access and liquidity logic
- risk analysis and limitations
- role of DeFi and blockchain infrastructure
